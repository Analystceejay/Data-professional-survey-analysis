## 📌 Project Overview

This project transforms raw survey data from 630 data professionals into a structured, insight-driven Power BI dashboard. All data connection, transformation, cleaning, and visualisation was handled entirely within **Power BI** — from loading the raw dataset through to the final interactive dashboard.

It answers six core business questions about compensation, career accessibility, tooling preferences, geographic distribution, and professional satisfaction — producing findings that are directly actionable for employers, aspiring data professionals, and training providers.

>  This project demonstrates the full power of Power BI for end-to-end business intelligence delivery.

## ❓ Business Questions Answered

This project is structured around **6 core business questions**:

1. **Which data roles command the highest average salaries?**
2. **How difficult is it to break into the data industry?**
3. **Which programming languages are most favoured by data professionals?**
4. **Where in the world are data professionals concentrated?**
5. **How satisfied are data professionals with their work-life balance?**
6. **How satisfied are data professionals with their salaries?**

Each question is answered with supporting data, visual evidence, and strategic recommendations in the [full report](report/Survey_Analysis_Data_Professionals_Report.docx).

## 🎯 Decisions & Actions Informed by This Analysis

The dashboard wasn't just built to display data — it was designed to drive real decisions. Here's what the findings directly informed:

| Insight from Dashboard | |
|---|---|
| Data Scientists earn ~$90K+ vs $50K for Analysts | Prioritise upskilling toward Data Science or Engineering for maximum salary growth |
| 31.75% found breaking in difficult or very difficult | Build and share a structured portfolio — proof of work matters more than credentials alone |
| Python dominates across every role | Focus all language learning on Python before branching into R or others |
| Salary happiness sits at 4.27/10 — below neutral | Professionals should actively benchmark their pay and negotiate; employers should audit compensation bands |
| Work/life balance scores 5.74/10 — above neutral but weak | Signals that remote/flexible work policies are a retention lever, not just a perk |
| USA leads, India is second, UK is third | Remote job searches should prioritise US-based roles; India and UK are strong secondary markets |
| Students/job seekers report the lowest salaries | Entry-level strategy should target Analyst roles as a stepping stone, not an endpoint |

> These decisions reflect how the dashboard translates survey data into career strategy, hiring policy, and training design.

## 💡 Strategic Recommendations (Summary)

**For Employers:**
- Review and benchmark Data Scientist compensation — the 4.27 salary satisfaction score signals a market-wide gap
- Improve work-life balance policies — a 5.74 score creates measurable retention risk
- Treat Python proficiency as a baseline hiring requirement across all data roles

**For Aspiring Data Professionals:**
- Invest in Python first — it dominates every surveyed role
- Build a public project portfolio to overcome the entry difficulty barrier

**For Educators & Training Providers:**
- Centre curricula on Python, statistics, and data visualisation
- Design portfolio-based entry programmes for career changers

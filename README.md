📌 Project Overview
This project transforms raw survey data from 630 data professionals into a structured, insight-driven Power BI dashboard. It answers six core business questions about compensation, career accessibility, tooling preferences, geographic distribution, and professional satisfaction — producing findings that are directly actionable for employers, aspiring data professionals, and training providers.
>  This project demonstrates the full power of Excel + Power BI for end-to-end business intelligence.

❓ Business Questions Answered
This project is structured around 6 core business questions:
Which data roles command the highest average salaries?
How difficult is it to break into the data industry?
Which programming languages are most favoured by data professionals?
Where in the world are data professionals concentrated?
How satisfied are data professionals with their work-life balance?
How satisfied are data professionals with their salaries?
Each question is answered with supporting data, visual evidence, and strategic recommendations in the full report.

---
💡 Strategic Recommendations (Summary)
For Employers:
Review and benchmark Data Scientist compensation — the 4.27 salary satisfaction score signals a market-wide compensation gap
Improve work-life balance policies — a 5.74 score creates measurable retention risk
Treat Python proficiency as a baseline hiring requirement across all data roles
For Aspiring Data Professionals:
Invest in Python first — it dominates every surveyed role
Target Data Science or Data Engineering for maximum salary upside
Build a public project portfolio to overcome the entry difficulty barrier
For Educators & Training Providers:
Centre curricula on Python, statistics, and data visualisation
Design portfolio-based entry programmes for career changers
Expand into India and UK markets — both are large, active data talent communities
